# Processed Data

- `nbs_crop_yield_state_zone_2022_2023.csv`: tidy state × crop × season yield table parsed from NBS NASS report tables.
- `state_metadata.csv`: state centroids and coarse agroecological-zone labels.
- `modeling_dataset.csv`: generated by `scripts/03_build_features.py`.

Rows with `is_aggregate=True` are zone/national totals and should normally be excluded from training.
